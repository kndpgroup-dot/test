# Quick Setup Guide - Bleame Shopify Theme

## 📦 What You Have

- **bleame-shopify-theme.zip** - Ready-to-upload Shopify theme package
- Location: `/app/bleame-shopify-theme.zip`

## 🚀 Upload to Shopify (3 Easy Steps)

### Step 1: Download the Theme File
The theme file is located at: `/app/bleame-shopify-theme.zip` (25KB)

### Step 2: Upload to Shopify
1. Log in to your Shopify admin at `yourstore.myshopify.com/admin`
2. Navigate to: **Online Store** → **Themes**
3. Click the **Add theme** button
4. Select **Upload zip file**
5. Choose `bleame-shopify-theme.zip`
6. Wait for upload to complete

### Step 3: Publish Your Theme
1. Once uploaded, click **Customize** to preview and edit
2. When ready, click **Actions** → **Publish** to make it live

## ⚙️ Initial Configuration

### 1. Homepage Setup
Go to **Theme Customization** and configure:

**Hero Section:**
- Upload hero image
- Edit headline: "Easy, Painless Hair Removal"
- Set button link to your main product or collection

**Product Badges:**
- Default badges are already set (Skin Safe, Reusable, Travel Friendly, Easy Clean)
- Customize icons and text as needed

**Featured Collection:**
- Create a collection first (Products → Collections → Create collection)
- Select this collection in the Featured Collection section
- Add at least 4 products for best appearance

**Testimonials:**
- Add 3 customer reviews with:
  - Review title
  - Review text
  - Customer name
  - Optional: Upload customer photo

**Stats Banner:**
- Edit the three percentage stats
- Customize the call-to-action button

**FAQ:**
- Add frequently asked questions
- Each item has a question and answer field

### 2. Create Products
Go to **Products** → **Add product**

Required fields:
- **Title**: e.g., "Crystal Hair Eraser"
- **Description**: Full product description
- **Price**: Regular selling price (e.g., $26)
- **Compare at price**: Original price for showing discounts (e.g., $44)
- **Images**: Upload 4+ high-quality images
- **Inventory**: Set quantity

### 3. Create Collections
Go to **Products** → **Collections** → **Create collection**

Suggested collections:
- "All Products"
- "Featured Products"
- "Best Sellers"

### 4. Create Pages
Go to **Online Store** → **Pages** → **Add page**

Required pages:
1. **Reviews** (template: page)
   - Title: "Customer Reviews"
   - Add review content
   
2. **Help Center** (template: page)
   - Title: "Help Center"
   - Add FAQ and support information

### 5. Configure Navigation
Go to **Online Store** → **Navigation**

**Main Menu** should include:
- Shop (link to /collections/all)
- Reviews (link to /pages/reviews)
- Help Center (link to /pages/help-center)

## 📞 Need Help?

Common issues and solutions:

**Q: Products not showing on homepage?**
A: Make sure you've selected a collection in the Featured Collection section settings

**Q: Cart not working?**
A: Ensure JavaScript is enabled and theme is properly published

**Q: Images not loading?**
A: Check image file sizes (should be under 1MB each) and formats (JPG, PNG, WebP)

**Q: Theme looks broken after upload?**
A: Try these steps:
1. Clear browser cache
2. Try a different browser
3. Check if theme was fully uploaded (no errors during upload)
