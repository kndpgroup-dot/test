# Οδηγός Γρήγορης Εγκατάστασης - Bleame Shopify Theme (Ελληνικά)

## 📦 Τι Έχετε

- **bleame-shopify-theme-greek.zip** - Έτοιμο Shopify theme με ελληνικά
- Τοποθεσία: `/app/bleame-shopify-theme-greek.zip`

## 🚀 Μεταφόρτωση στο Shopify (3 Απλά Βήματα)

### Βήμα 1: Κατεβάστε το Αρχείο Theme
Το αρχείο βρίσκεται στο: `/app/bleame-shopify-theme-greek.zip` (28KB)

### Βήμα 2: Μεταφόρτωση στο Shopify
1. Συνδεθείτε στο Shopify admin στο `yourstore.myshopify.com/admin`
2. Πλοηγηθείτε σε: **Online Store** → **Themes**
3. Κάντε κλικ στο κουμπί **Add theme**
4. Επιλέξτε **Upload zip file**
5. Επιλέξτε το `bleame-shopify-theme-greek.zip`
6. Περιμένετε να ολοκληρωθεί η μεταφόρτωση

### Βήμα 3: Δημοσιεύστε το Theme σας
1. Μόλις μεταφορτωθεί, κάντε κλικ στο **Customize** για προεπισκόπηση και επεξεργασία
2. Όταν είστε έτοιμοι, κάντε κλικ στο **Actions** → **Publish** για να το κάνετε ζωντανό

## ⚙️ Αρχική Διαμόρφωση

### 1. Ρύθμιση Αρχικής Σελίδας
Πηγαίνετε στο **Theme Customization** και διαμορφώστε:

**Τμήμα Hero:**
- Μεταφορτώστε εικόνα hero
- Επεξεργαστείτε τον τίτλο: "Εύκολη, Ανώδυνη Αποτρίχωση"
- Ορίστε σύνδεσμο κουμπιού στο κύριο προϊόν ή συλλογή σας

**Σήματα Προϊόντων:**
- Τα προεπιλεγμένα σήματα είναι ήδη ορισμένα
- Προσαρμόστε εικονίδια και κείμενο όπως χρειάζεται

**Προβεβλημένη Συλλογή:**
- Δημιουργήστε πρώτα μια συλλογή (Products → Collections → Create collection)
- Επιλέξτε αυτή τη συλλογή στο τμήμα Προβεβλημένη Συλλογή
- Προσθέστε τουλάχιστον 4 προϊόντα για καλύτερη εμφάνιση

**Κριτικές Πελατών:**
- Προσθέστε 3 αξιολογήσεις πελατών με:
  - Τίτλο αξιολόγησης
  - Κείμενο αξιολόγησης
  - Όνομα πελάτη
  - Προαιρετικό: Μεταφορτώστε φωτογραφία πελάτη

**Μπάνερ Στατιστικών:**
- Επεξεργαστείτε τα τρία ποσοστά στατιστικών
- Προσαρμόστε το κουμπί κλήσης σε δράση

**Συχνές Ερωτήσεις:**
- Προσθέστε συχνές ερωτήσεις
- Κάθε στοιχείο έχει ένα πεδίο ερώτησης και απάντησης

### 2. Δημιουργία Προϊόντων
Πηγαίνετε στο **Products** → **Add product**

Απαιτούμενα πεδία:
- **Τίτλος**: π.χ., "Crystal Hair Eraser"
- **Περιγραφή**: Πλήρης περιγραφή προϊόντος
- **Τιμή**: Κανονική τιμή πώλησης (π.χ., €26)
- **Τιμή σύγκρισης**: Αρχική τιμή για εμφάνιση εκπτώσεων (π.χ., €44)
- **Εικόνες**: Μεταφορτώστε 4+ εικόνες υψηλής ποιότητας
- **Απόθεμα**: Ορίστε ποσότητα

### 3. Δημιουργία Συλλογών
Πηγαίνετε στο **Products** → **Collections** → **Create collection**

Προτεινόμενες συλλογές:
- "Όλα τα Προϊόντα"
- "Προβεβλημένα Προϊόντα"
- "Μπεστ Σέλερ"

### 4. Δημιουργία Σελίδων
Πηγαίνετε στο **Online Store** → **Pages** → **Add page**

Απαιτούμενες σελίδες:
1. **Αξιολογήσεις** (πρότυπο: page)
   - Τίτλος: "Αξιολογήσεις Πελατών"
   - Προσθέστε περιεχόμενο αξιολόγησης
   
2. **Κέντρο Βοήθειας** (πρότυπο: page)
   - Τίτλος: "Κέντρο Βοήθειας"
   - Προσθέστε συχνές ερωτήσεις και πληροφορίες υποστήριξης

### 5. Διαμόρφωση Πλοήγησης
Πηγαίνετε στο **Online Store** → **Navigation**

Το **Κύριο Μενού** πρέπει να περιλαμβάνει:
- Αγορά (σύνδεσμος στο /collections/all)
- Αξιολογήσεις (σύνδεσμος στο /pages/reviews)
- Κέντρο Βοήθειας (σύνδεσμος στο /pages/help-center)

- Εικονίδιο λογαριασμού χρήστη

- Εμφανίζει σήματα εμπιστοσύνης (Δωρεάν Αποστολή, Εγγύηση 30 Ημερών, Εύκολες Επιστροφές)
